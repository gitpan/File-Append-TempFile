File::Append::TempFile — append to files using a temporary copy

Welcome to the home of the File::Append::TempFile Perl module — a simple way to append data to files in an almost secure, almost atomic, almost reliable fashion :) The most up-to-date version of this webpage should be available at the Ringlet development webserver.

Anyway, if you really want to know what File::Append::TempFile is or does, there is the HTML version of its documentation.

The current release of File::Append::TempFile is 0.04; it is available for download in several different flavors:

  • Source: File-Append-TempFile-0.04.tar.gz (PGP signed)
  • FreeBSD: there is a port, devel/p5-File-Append-TempFile, which is probably available at your friendly local CVSup server :)
  • Debian: an i386 package (File-Append-TempFile-0.04-1), along with its source components: dsc, changes, diff, orig.

Also, for your downloading and comparing enjoyment, we provide some oldish sysgather versions:

  • File-Append-TempFile-0.03
    • Source: File-Append-TempFile-0.03.tar.gz (PGP signed)
    • Debian: an i386 package (File-Append-TempFile-0.03-1), along with its source components: dsc, changes, diff, orig.
